Welcome to the New Zealand Sign Language Web


Explore and learn New Zealand Sign Language, NZSL (Te Reo Turi) with our easy-to-use application!

The web app aims to design and develop a web application to assist beginners in learning NZSL.


  1. Home page


    Displays a brief description of this page and the version of the server.

  2. NZSL


    Shows all the different NZSL signs and their meanings.

    This page also has a search bar option!


  3. Events


    This section shows to the user a list of events as supplied by the server.

    The user is able to add selected events to your favourite calendar application!
